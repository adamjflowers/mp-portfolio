Design Log Week One


Posted by Adam Flowers on November 15th, 2019

Monday 11th November

What did I learn and why?

I learnt the basic principles of web design and looked at examples of responsive design. I also learnt more about the difference between web developers, designers and researched examples of websites themes/styles that I could use on my own site. We began our photography course by being introduced to the equipment we will be using and learning some of the basic principles of photography.

How will I implement this learning into my final portfolio?

I will use the examples and information I learnt to inform my website design choices. Being familiar with the camera will help me down the line when I begin to take photographs for my portfolio.

Tuesday 12th November

What did I learn and why?

I was introduced to using Github alongside the code editor Atom and built my first respiratory using a bootstrap theme. I made my first changes to my site using atom and then pushing to origin and committing changes using the Github desktop app.  During photography we learnt some photographic techniques such as the rule of thirds and  depth of vision.

How will I implement this learning into my final portfolio?

I will use the skills I learnt on Atom code editor to make further changes to improve my website. My new knowledge of Github will help me continue to maintain and change my website

Wednesday 13th November

What did I learn and why?

For Digital Wednesday I was tasked with reading the Dieter Rams Design Principles. This reading enabled me to learn some web design principles that I could incorporate into my own website. I researched websites which were good examples of these principles. I found examples of the principles of “good design is honest design.”

How will I implement this learning into my final portfolio?

These principles will help make sure that the changes I make to the site are suitable and good design choices.

Thursday 14th November

What did I learn and why?

In web design we learnt about information architecture   we created several pages and then linked them together, the prototype website now has 5 fully linked pages. During photography we had a guest lecture by Ryan from Salford City. Who spoke to use about social media use by the club and the importance of composition when making content.

How will I implement this learning into my final portfolio?

I will use the knowledge I gained to make sure that my website has linked pages that function fully. I can take lots of useful advice from hearing Ryan talk, I will make sure the composition of my final images is correct.  

Friday 15th November

What did I learn and why?

We learnt about typography in web design and changed the fonts on our website.  In photography we began to start taking photographs using the techniques we have learnt so far, we took 5 photographs demonstrating 5 different photographic techniques.

How will I implement this learning into my final portfolio?

I will make sure I use different and suitable fonts throughout my website. Serif for titles and sans serif for the body text for example. I can use the different photographic techniques I learnt to take high quality images for my final portfolio.
